backend/core/config.py
  Type:           Source — Configuration
  Classification: INTERNAL
  Description:    Pydantic Settings class for centralized environment
                  management. Reads all configuration from environment
                  variables. Settings include: API_KEY, AOAI_ENDPOINT,
                  Azure OpenAI key, Cosmos DB connection string,
                  MOCK_AI flag, and all other environment-driven
                  configuration. No default values for secrets
                  (raises ValidationError if required env vars
                  are missing).
  Dependencies:   pydantic-settings, python-dotenv
  Status:         Present

backend/core/security.py
  Type:           Source — Security
  Classification: INTERNAL
  Description:    API authentication implementation. Contains the
                  api_key_guard() FastAPI dependency function.
                  Reads API_KEY from settings and validates the
                  X-API-Key request header. Returns HTTP 401 on
                  invalid or missing key.
  Implementation:
    from fastapi import Header, HTTPException, status, Depends
    from .config import settings
    def api_key_guard(x_api_key: str = Header(...,
        alias="X-API-Key")) -> None:
        if not settings.API_KEY or x_api_key != settings.API_KEY:
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="Invalid API key"
            )
  Security Note:  Minimal, auditable implementation. No bypass logic.
  Status:         Present
